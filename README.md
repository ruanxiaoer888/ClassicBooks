## 📖 使用说明

### 文件特点
- **格式统一**: 所有书籍均为PDF格式，便于阅读和保存
- **分类清晰**: 按照主题分类，便于查找相关书籍
- **版本标注**: 部分书籍标注了版本信息，如[V]表示特定版本
- **作者信息**: 书名后标注了作者信息，便于识别
- **LFS管理**: 使用Git LFS管理大文件，确保版本控制效率

### 查找方式
- **按主题浏览**: 根据兴趣领域选择相应文件夹
- **关键词搜索**: 使用文件管理器的搜索功能查找特定书籍
